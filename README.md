# RAKSHAK-EDR 🛡️

SOC-style Endpoint Detection & Response Dashboard built using Python, Flask, SQLite, and Scapy.

## Features
- Live packet monitoring
- Threat severity alerts
- Login authentication
- Threat logs dashboard
- CSV export support
- Cyberpunk SOC UI

## Tech Stack
- Python
- Flask
- SQLite
- Scapy
- HTML/CSS/JavaScript

## Project Purpose
Built for learning SOC workflows, endpoint monitoring, and cybersecurity dashboard development.
